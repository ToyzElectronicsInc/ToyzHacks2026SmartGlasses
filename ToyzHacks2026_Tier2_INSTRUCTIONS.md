# Tier 2 — Unity DLC Component

Place your Unity assets in a folder named after your GitHub username:

```
Tier2/
  your-github-username/
    YourComponentName.prefab
    YourComponentDataSO.asset
    Scripts/
      YourComponentBehaviour.cs
    README.md
```

Unity version: 6.3.7f1. Follow ScriptableObject architecture rules in the main README.
